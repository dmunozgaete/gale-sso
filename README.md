# Worker Manager - Web Application

- Web Application (Gale + Angular Gale + Angular Gale Material)
- API (Worker Cron Manager)


## Resources 

- [API](https://bitbucket.org/desarrollo_sodimac/api-worker-manager)
- [Angular Gale Doc's](http://angular-gale.azurewebsites.net/)
- [Framework API](http://gale.azurewebsites.net/)

## Get Started

### Get dependencies

```sh
# bash
npm install				# restore npm packages
bower install			# restore bower libraries
```

### Mount development server

```sh
# bash
grunt lift			# Mount web server without [livereload]
```

### Build for production stage

```sh
# bash
grunt deploy		 # Deploy and Minify Files
```
